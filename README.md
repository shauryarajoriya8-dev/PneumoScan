# PneumoScan
AI Pneumonia Detection Tool 
• Developed a deep learning-based pneumonia diagnostic tool using VGG16 CNN, achieving 92% accuracy on
X-ray scans. Integrated Grad-CAM heatmaps for model interpretability, TTS alerts, and automated PDF
reporting
• Trained on Kaggle’s Chest X-Ray Pneumonia dataset; key features include AI diagnosis, heatmap
visualization, TTS-based results, doctor consultation support, and report generation
